CQ4OE Benchmark

User Guide GitHub
A benchmark for assessing LLM-assisted ontology generation from competency questions

A reproducible benchmark for evaluating LLM-generated ontologies against CQ-aligned gold standards

Six source ontologies with explicit provenance from each competency question to the classes, properties, and axioms required to answer it. Two evaluation tasks. CQ2Term measures CQ-specific term prediction over 99 CQs. CQ2Onto measures full-ontology generation over 118 CQs across five targets.
